账单发送邮件规范

集中查看各账户的发件邮箱、账户号码口径、邮件主题及中英文正文。页面以原 Word 文档中的“邮件内容”为基础，并纳入业务已确认口径。

使用提示：邮件正文按纯文字展示；方括号内容为系统需要替换的变量。复制前请先确认结单周期、客户姓名、账户号码和日期。

发送配置速查

按账户类型确认发件邮箱与正文中的账户号码口径。

邮件模板

选择账户和周期后，可分别复制主题或完整正文。

已确认口径

以下内容已根据业务确认更新至邮件主题及正文。

邮件正文统一为纯文字；英文拼写及 Account Type 继续按当前版本保留。

  • 示例日期带连接符非标准账户日结单示例使用 2026-01-01，月结单示例使用 2026-01，与主题变量格式一致。
  • 正文账户 ID 已明确第一个 ID 为 [Derivatives Account ID]；TRS 模板的第二个 ID 为 [TRS Account ID]。
  • 公司名称已确定投资类模板免责声明使用 PrincipleOne Investments Limited。
  • 英文拼写暂不调整MAGRIN 与 RETUEN 按当前版本保留。
  • 账户类型保持不变期权、共赢及 TRS 正文中的 Account Type 继续使用 Derivatives Account。
  • 标准账户资料日期格式结单日期统一使用 YYYY-MM-DD。
